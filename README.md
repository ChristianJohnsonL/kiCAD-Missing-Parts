# kiCAD-Missing-Parts
An open-source collection of KiCad assets for components that don't exist on UltraLibrarian or SnapMagic. All designs verified against manufacturer datasheets. Compatible with:  KiCad Altium Designer Autodesk EAGLE EasyEDA / JLCPCB EDA DipTrace (STEP import): SolidWorks Autodesk Fusion  Onshape  FreeCAD
